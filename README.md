# Campus Delivery API

Backend API for a campus food delivery system with authentication and role-based access.

## Features
- JWT authentication
- Role-based access (admin, customer, rider)
- Restaurants & menu management
- Order creation and tracking
- PostgreSQL + Prisma ORM
- Swagger API documentation

## Tech Stack
- Node.js
- Express
- PostgreSQL
- Prisma
- JWT
- Swagger

## Setup

```bash
git clone <repo-url>
cd campus-delivery-api
npm install
